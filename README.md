# EcoBottle Shop

A responsive, professional e-commerce website showcasing reusable water bottles. Built using **HTML** and **CSS** only.

## Features

- Responsive layout for desktop and mobile.
- Hero section with blended cover image.
- Navbar with working hamburger menu on small screens.
- 6 product cards with images, price, and review stars.
- About and Contact sections.
- Footer with copyright.

## Technologies

- HTML5
- CSS3
- Google Fonts (Montserrat)

## Notes

- All images should be downloaded locally for faster loading.
- Navbar links scroll to respective sections using `id`.
